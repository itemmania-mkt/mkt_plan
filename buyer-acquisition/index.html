Strategy Overview

한 장 요약

구매자 모객 전략을 고객 접점 확대 → 사이트 방문 → 구매 전환 → 재방문 흐름으로 연결해, 아래 4대 방향을 중심으로 추진하고자 합니다.

추진 방향 4가지
① 고객 접점 직접 발굴 — 검색·커뮤니티·영상·제휴에서 마케팅팀이 신규 접점을 직접 확보
② 방문 이유 제공 — 시세·적정가, 판매자 신뢰 정보, 원하는 매물을 구매 단계 전면에 노출
③ 거래·재방문 유도 — 첫 구매 지원, 찜·가격 알림, 구매 휴면 CRM 운영
④ 단일 게임 의존도 완화 — 성장 게임의 매물·콘텐츠 확대로 거래 기반 분산
판단 근거
거래금액 급락의 96.5%가 리니지클래식 단일 게임에서 발생했고, Google 자연검색은 순위·CTR이 개선됐음에도 노출·클릭이 감소해 관련 검색 수요 자체의 감소가 확인됩니다. 즉 플랫폼 경쟁력 문제라기보다 단일 게임 의존과 내부 전환 체계 부족이 원인이므로, 광고 집행 확대만으로는 회복이 어렵다고 판단했습니다.

Part 2 · 전략 방향

문제 재정의 — "모객"에서 "유동성·전환"으로

외부 광고 확대 전에 내부 누수를 막고, 구매자와 판매자를 같은 시장 단위에서 동시에 운영해야 합니다.

기존 관점 → 재정의
"구매자를 더 데려오는 방법" → "구매 의도가 생긴 이용자가 원하는 매물을 발견하고, 응답 가능한 판매자와 거래를 완료한 뒤, 다시 구매하도록 만드는 방법"

구매자 이탈이 발생하는 5개 구간

구간진단 내용대응 전략축
① 유입 부족구매 의도가 있는 게이머에게 브랜드·콘텐츠가 도달하지 않음축 A 유입 / 축 E 콘텐츠
② 검색 실패게임·서버·가격 조건에 맞는 매물이 없음축 D 공급 / 축 A 유입
③ 신뢰·속도 이탈판매자 미응답, 신뢰정보 부족, 먹튀 불안축 B 전환 / 축 D 공급
④ 신청·결제 이탈UX·인증·결제오류 또는 가격 부담으로 중단축 B 전환 / 축 C 유지
⑤ 재구매 실패구매주기·게임 이벤트에 맞춘 후속 관리 부족축 C 유지

구매자의 플랫폼 선택 기준과 대응 방향

선택 기준구매자 관점아이템매니아 대응 방향
가격·시세동일 게임·서버·상품의 적정 가격을 비교시세 정보·적정가 판단 지원 강화
매물·선택권원하는 조건의 상품을 빠르게 탐색성장 게임·핵심 서버 매물 확대
안전·신뢰판매자 이력과 거래 안전성을 확인안전 정보·거래 이력을 구매 단계에서 명확히 노출
편의성가입·검색·결제·거래 완료 과정의 간편함첫 구매 경로 단축·안내 강화
정보·콘텐츠시세·거래방법·업데이트·사기예방 정보 탐색검색·블로그·커뮤니티·영상 콘텐츠 연결
포지셔닝
"다양한 게임의 매물을 비교하고 안전하게 거래할 수 있는 대표 게임 거래 플랫폼" — 가격 경쟁만 강조하기보다 선택권·안전성·판단 정보를 함께 제공하는 대중적·안정적 플랫폼으로 포지셔닝합니다.
북극성 지표 제안
단순 방문자·신청자가 아니라 기간 내 실제 구매확정을 완료한 고유 구매자 수(Successful Buyers)를 중심지표로 두고, 게임·서버별 활성시장 수·재구매율·거래완료시간을 함께 관리합니다.
Part 2-1 · 고객 접점과 방문 이유

고객을 기다리는 마케팅에서, 고객을 찾아가는 마케팅으로

과거에는 PC방 방문, 홍보물 배포, 현장 영업을 통해 고객이 있는 곳에서 브랜드를 직접 알렸습니다. 현재 고객의 활동 공간은 검색, 영상, 커뮤니티, AI 검색 등 온라인으로 이동했으며, 마케팅팀도 그 접점으로 직접 이동해야 합니다.

과거 고객이 있는 곳으로 직접 이동

PC방·게임 행사·홍보물·현장 제휴
  • 직접 접점 확보 — 고객과 대면하고 반응을 확인
  • 반복 노출 — 생활 동선 안에서 브랜드 인지도 형성
  • 현장 영업 — 마케팅팀이 직접 고객을 발굴

현재 온라인 고객 접점으로 전환

Google·네이버·YouTube·커뮤니티·AI 검색·디스코드
  • 검색 접점 — 게임명·시세·거래방법·복귀·서버이전 키워드 선점
  • 커뮤니티 접점 — 정보형 콘텐츠와 현장 반응 수집
  • 영상 접점 — 업데이트·복귀·신작 이슈에 맞춘 즉시 노출
  • 직접 영업 — 커뮤니티 운영자·스트리머·PC방·게임 행사와 직접 제휴
마케팅팀의 역할
대행사는 실행 수단으로 활용하되, 고객이 어디에 있고 무엇을 필요로 하는지 직접 찾아내는 역할은 내부 마케팅팀이 맡아야 합니다. 마케팅팀을 광고 운영 조직이 아니라 구매자를 발굴하고 사이트 방문으로 연결하는 세일즈 조직으로 전환하고자 합니다.

구매자가 사이트를 방문해야 하는 이유

구매자 니즈방문 이유강화 방향
믿을 수 있는 거래판매자와 거래 안전성을 빠르게 확인거래 이력·평점·응답률·검증 판매자·사고보상 정보를 상품 상단에 노출
적정 가격 판단가격이 비싼지 싼지 확인게임·서버별 시세, 적정가, 가격 추이, 유사 매물 비교 제공
원하는 매물 탐색조건에 맞는 매물을 빠르게 찾음희망매물 등록, 가격 알림, 신규 매물 알림, HOT 매물 큐레이션
빠른 거래응답 가능한 판매자와 즉시 거래판매자 최근 접속·응답률·거래 가능 상태 표시
거래 정보거래방법·주의사항·게임별 특징 확인시세·가이드·FAQ·업데이트 영향·사기 예방 콘텐츠를 게임별 정보 허브로 제공

구매자 여정과 마케팅 개입 지점

1. 발견고객이 정보를 찾는 순간
  • Google·네이버·AI 검색 노출
  • 유튜브·커뮤니티·디스코드 접점
  • 신작·복귀·서버이전·업데이트 이슈 대응
2. 방문·탐색사이트에서 판단하는 순간
  • 시세·가격 비교·추천 매물
  • 판매자 신뢰 정보·안전거래 안내
  • 희망매물·가격알림 등록
3. 거래·재방문구매를 반복하게 만드는 순간
  • 첫 구매 경로 단축·혜택 안내
  • 찜·알림·휴면 CRM
  • 게임 이벤트와 구매주기 기반 재방문
서비스 적용 방향
그룹 내 해외 거래 플랫폼에서 확인되는 강점은 별도 사례로 소개하기보다 국내 서비스의 강화 항목으로 반영합니다. 핵심은 판매자 신뢰도 가시화, 상품 정보 표준화, 거래 보호 체계 전면 노출, 게임별 정보 허브, 구매 알림·추천 기능입니다.

Part 4 · 취합 결과

6대 전략축 통합 지도

중복·유사 아이디어를 하나로 묶어 6대 전략축으로 재구성했습니다.

A 유입 — 검색·SEO·PR

구매 의도가 생기는 순간을 선점
  • SEO 전면 재정비 + 게임×아이템×서버 시세페이지 대량 생성
  • 시세 데이터 API·PR 저널리즘 — 시세 위젯 매체 무료 제공, 기사화로 무광고 유입
  • 고의도 검색광고 + 게임별 전용 랜딩 (게임명+재화+시세 키워드)

B 전환 — 첫 구매·신뢰

방문을 첫 거래로 빠르게 전환
  • 첫구매 허들 완화 + 메인 [첫구매혜택] 버튼 노출
  • 상품 리포트 최상단 '안전 지표' 배치 — 판매자 거래 성공률·응답률·최근 접속, 거래이력, 사고보상 여부를 가격보다 먼저 노출
  • 구매자 보호 체계 전면 노출 — 사고보상·에스크로·분쟁 대응 절차를 구매 단계에서 안내
  • 계정 스펙 수치화·표준화 — 게임별 스펙 항목 표준화로 가격 적정성 판단과 상품 비교 시간 단축
  • 가입 후 7일 첫 구매 CRM 캠페인 (0·1·3·7일 메시지)

C 유지 — CRM·세그먼트·알림

이미 방문한 구매자의 손실 최소화
  • 찜·알림 레이어 고도화 — 가격변동·매칭매물 능동 알림
  • 구매 휴면 CRM (30·60·90일 세그먼트 재구매 유도)
  • 구매·결제 이탈 복구 (미완료·취소·미응답 원인별)
  • 세그먼트별 개인화 노출 (배너·카피·매물 차등)

D 공급 — 매물·판매자 유동성

실제 거래 가능한 매물과 판매자 확보
  • 게임·서버별 유동성 대시보드 + 광고집행 게이트
  • 타 인기 게임 파워셀러 직접 유치 (수수료 할인)
  • 리니지클래식 맞춤 등록 UI + HOT매물 자동 분류

우선 실행 TOP 10 과제

효과·난이도·실행 시점을 종합한 추천 순서입니다. 상위 5개는 개발 부담이 낮아 즉시 착수가 가능합니다.

순위실행 과제핵심 대상실행안난이도시점
1찜·알림 고도화구매대기 고객찜한 상품의 가격 하락·원하는 매물 등록 시 자동 알림 → 재방문 유도 (업계 재입고 알림의 방문·전환율이 매우 높음)낮음즉시
2구매 휴면 CRM30·60·90일 미구매미구매 회원에게 과거 구매 게임·금액대 맞춤 메시지 발송 → 재구매 유도. 일부는 미발송 대조군 대비 효과 검증낮음즉시
3구매·결제 이탈 복구구매 도중 이탈구매 중단 고객(미결제·취소·판매자 미응답)에게 원인별 대체 매물·결제 방법 안내 → 거래 복구낮음즉시
4첫구매 전환 패키지가입 후 미구매메인에 첫구매 혜택 버튼·안전 정보(사고보상 등) 노출 + 가입 후 7일 단계별 안내. 대상자에게만 첫 구매 혜택 이벤트 플로팅 배너 노출낮음즉시
5바이럴·SNS 할인쿠폰신규 잠재 고객블로그·SNS 콘텐츠에 자체 발급 할인쿠폰 결합 → 콘텐츠 조회를 첫 거래로 전환낮음~중간즉시
6SEO·시세 랜딩 재정비검색 유입 신규'게임명+시세·거래' 검색 상위 노출용 페이지 정비 + 게임×서버별 시세 페이지 확대 → 광고비 없이 유입 확대중간단기
7수요·공급 통합 캠페인신작·업데이트 시점수요가 몰리는 신작·업데이트 시점에 구매자·판매자 동시 모집. 신작은 72시간 집중 혜택·전용 이벤트 페이지로 즉각 대응중간단기
8추천인·리퍼럴기존·신규 구매자친구의 '가입'이 아닌 '첫 실제 거래' 완료 시 보상 → 실구매자만 유입. 자기추천·중복 가입 등 어뷰징 차단중간단기
9조기경보·공급 대시보드내부 운영게임·서버별 거래액·매물·판매자 통합 모니터링 도구. 특정 게임 급감(전주 −15%↑) 시 자동 알림으로 즉시 대응, 매물·판매자 부족 게임엔 광고 보류로 예산 낭비 방지 (리니지클래식 급락 뒤늦은 발견 문제 재발 방지)중간단기
10타게임 파워셀러 유치이탈위험·공급성장 게임(메이플·로스트아크)의 우수 판매자 유치·매물 확대 → 리니지클래식 단일 게임 쏠림 위험 완화중상중기

각 채널은 개별 노출이 목적이 아니라, 게임별 랜딩과 실제 구매 행동으로 연결되도록 설계합니다.

채널실행 방안핵심 타깃
SEO·검색주요 게임 페이지 구조 개선, 시세·거래 가이드 확대, 게임별 랜딩 연결시세·거래 방법 검색 고객
사이트·랜딩첫 구매 혜택 노출, 모바일 경로 단축, 시세·안전 정보 상품 상단 배치신규·구매대기 고객
CRM가입 미구매·취소·미완료·미재구매 행동별 메시지, 최근 구매 게임 기반 개인화신규·기존·이탈회원
블로그·콘텐츠시세·가이드·계정 가치 콘텐츠 제작, 상품·게임 페이지 직접 연결잠재 구매자
커뮤니티게임별 정보형 콘텐츠 운영, 과도한 홍보 배제, 이용자 반응·이슈 모니터링게임별 고관여 이용자
유튜브·인플루언서실사용 리뷰·쇼츠·길드 연계 콘텐츠, 전용 링크·코드로 거래 연결팬덤·길드 이용자
SNS·밈트렌드 적합 시 간헐 운영, 안전거래·시세 정보 메시지 결합젊은 잠재 고객
이벤트·추천첫 유효 구매 완료 기준 친구추천, 거래 횟수 기반 미션형 프로모션신규·반복 구매자
앱·알림가격 변동·신규 매물·게임 업데이트·쿠폰 혜택 알림구매대기·기존 구매자
데이터 운영게임별 거래 변화·구매 행동·이탈 징후를 정기 확인해 실행 과제에 반영내부 운영
Part 8 · 실행 로드맵

90일 단계별 실행 계획

분석 → 즉시 실험 → 고도화 → 확대 순으로 추진하며, 각 단계에 Go/No-Go 기준을 둡니다.

1단계 · 즉시 검증0 ~ 30일
  • 기준선 분석 + 조기경보·유동성 대시보드 착수
  • 찜·알림 레이어 고도화 착수(2주)
  • 구매 휴면 CRM 3종 발송(대조군 유지)
  • 구매·결제 이탈 복구 캠페인 개시
  • 첫구매 전환 패키지(버튼·안전지표·7일 CRM)
  • 바이럴·SNS 자체 할인쿠폰 발급
  • SEO·시세 랜딩 재정비 — 기본 정비(title/meta/H태그)
2단계 · 고도화31 ~ 60일
  • SEO·시세 랜딩 재정비 — 시세페이지 대량 생성·구조화
  • 수요·공급 통합 캠페인 1건 운영
  • 추천인·리퍼럴 도입(첫 유효거래 기준)
  • 타게임 파워셀러 유치·포트폴리오 다변화 착수
  • 활성 판매자 SLA·신규셀러 스프린트
  • 게임 이슈 레이더·코호트 시범 운영
3단계 · 확대·개발61 ~ 90일
  • 성과 매체·캠페인 확대, 저효율 종료
  • VIP 등급제·마일리지 설계
  • 리니지클래식 맞춤 등록 UI·HOT매물 분류
  • 시세 API·PR 저널리즘
  • 계정 스펙 수치화 개발 타당성 검토
  • 정례 업무 전환·개발 우선순위 확정

Go/No-Go: 1단계 데이터 신뢰성 확보 → 2단계 증분 구매·CAC 확인 → 3단계 수요충족·수익성 확인

Part 10 · 결론

최종 결론

구매자 모객은 신규 유입 확보와 내부 전환 개선을 동시에 추진해야 합니다. 앞서 제시한 4대 방향을 90일 로드맵 순서로 실행해, 단기 거래 회복과 안정적인 구매자 기반 확대를 함께 달성하고자 합니다.
